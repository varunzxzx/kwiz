# KWIZ - Online Quiz Portal

## Stack Used
- Frontend (React)
- Backend (Express)
- Databse (MongoDB)
- hosting (Heroku)
- React + Express combined

### Features
- React
- Express
- React-Router
- Webpack
- ES6 Support
- CSS-Loaders
- Hot Reload Module

#### To get started
```shell
$ git clone https://github.com/varunzxzx/kwiz.git
$ npm install
```
#### To start a development server
```shell
$ npm dev
```

#### To start a production server
```shell
$ npm start
```

#### To build the app
```shell
$ npm build:prod
```
